=============================================================
Beaconizer
=============================================================

.. only:: html

	:Version:
		|release|

	:Language:
		en

	:Description:
		Harvest links from BEACON file providers and offer your data as links via BEACON files. Features a scheduler job for harvesting BEACON files, a BEACON generator and a seeAlso plugin for content enrichment. `Check out Wikipedia to learn more about the BEACON format. <https://meta.wikimedia.org/wiki/Dynamic_links_to_external_resources>`_
